## Overview

A conversational AI system that provides empathetic, non-clinical emotional support using transformer-based NLP. The system detects the user's emotional state and sentiment in real-time and generates contextually appropriate, supportive responses.

**This tool is not a substitute for professional mental health care.**

---

## System Architecture

```
User Input
   ↓
Crisis Detection (keyword filter — highest priority)
   ↓ (if no crisis)
Emotion Classification  ←→  Sentiment Analysis
(j-hartmann/emotion-english-distilroberta-base)   (cardiffnlp/twitter-roberta-base-sentiment)
   ↓
Response Generation (rule-based, (emotion × sentiment) lookup table)
   ↓
Empathetic Reply → Streamlit Web UI
```

### 2. Install dependencies


## Models

| Component | Model | Task |
|-----------|-------|------|
| Emotion Classifier | `j-hartmann/emotion-english-distilroberta-base` | 6-class emotion detection |
| Sentiment Classifier | `cardiffnlp/twitter-roberta-base-sentiment` | 3-class polarity detection |

Both models are loaded via HuggingFace Transformers with singleton lazy loading for memory efficiency.

### Evaluation Components

✅ Emotion Classification (Anger, Disgust, Fear, Joy, Neutral, Sadness, Surprise)

✅ Sentiment Analysis (Positive, Negative, Neutral)

✅ Crisis Detection Module

✅ Empathetic Response Generation

✅ Conversation Logging System

### Observations

- The emotion classifier achieved strong performance with an accuracy of 74.73%.
- The sentiment classifier achieved an accuracy of 61.17%.
- Crisis-related inputs are intercepted before normal response generation to provide safety-focused responses.
- All interactions are logged with timestamps, emotion labels, sentiment predictions, confidence scores, and generated responses for analysis and future improvements.

## Ethical Design

- **Non-diagnostic**: No medical diagnosis or clinical advice is provided.
- **Non-persistent**: No user data is stored beyond the local session log.
- **Crisis-aware**: Crisis keyword detection triggers immediate referral to professional resources.
- **Transparent**: The system is explicitly identified as AI at all times.
